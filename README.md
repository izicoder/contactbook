### Small contact book SPA

#### Built with

-   Express.js;
-   Vite;
-   LowDB;
-   React.js;
-   Typescript;

#### Usage

Requires node.js version 22 and linux environment

How to run:

`git clone https://github.com/izicoder/contactbook`

`cd contactbook`

`./build.sh`

`(node build/backend/dist/index.js) & (npx serve build/frontend/dist)`
